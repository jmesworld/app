
# JMES Mobile Wallet App 

# Requirements 
   - Node.js version > 17.0.0
## Instructions

1. Install dependencies

   ```bash
   $ yarn or npm install
   ```

2. Run Application

   ```bash
    $ yarn start or npm run start
    ``` 
   
3. Choose the the platform you wish to launch the application [web|android|ios]

4. Enjoy!
